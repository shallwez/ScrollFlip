# 滚动翻转

一款原生 macOS 菜单栏应用，可单独调整物理鼠标的垂直滚轮方向和速度，不影响触控板。

## 功能

- 鼠标垂直滚动翻转开关
- 0.25×～3.0× 滚轮速度
- 自动识别并跳过触控板连续滚动事件
- 设置自动保存
- 菜单栏常驻，无 Dock 图标

## 构建与运行

```bash
chmod +x build-app.sh
./build-app.sh
open "dist/滚动翻转.app"
```

首次运行后，点击菜单栏鼠标图标，根据提示前往“系统设置 → 隐私与安全性 → 辅助功能”，允许“滚动翻转”。授权后重新打开应用即可。

开发调试也可以运行：

```bash
swift run ScrollFlip
```

> 注意：重新构建的临时签名可能导致 macOS 要求再次授权。正式分发时应使用稳定的 Apple Developer 证书签名并公证。
